# ayanTimer
:tomato:番茄工作法

```javascript
var timer = new AyanTimer({
    container: document.querySelector("#timer")  
})
```

## Getting started

```shell
npm install
```

## Development

```shell
npm run dev
npm run dev:web //web
```

## Packaging

To package the app for the local platform:

```shell
npm run package:win
```
